MySQL-数据合并

原数据：

name type money
jack 月薪 10000
jack 年终奖 3000
lisa 月薪 20000
lisa 年终奖 3500

期望数据：

name salary reword
jack 120000 3000
lisa 240000 3500

sql语句：

INSERT INTO emp(name, salary, reword)

​ SELECT

​ name,

​ GROUP_CONCAT(CASE WHEN type = ‘月薪’ THEN money*12 END),

​ GROUP_CONCAT(CASE WHEN type = ‘年终奖’ THEN money END)

​ FRPM emp

​ GROUP BY name;
